Srinithya Thammanaveni - Portfolio Website
A modern, professional portfolio website designed to showcase my skills, projects, and background as a B.Tech Computer Science student. This responsive single-page application serves as my digital resume and demonstrates my web development capabilities to potential employers, collaborators, and academic institutions.

Project Overview
This portfolio website is a complete personal branding solution that presents my professional identity in an engaging, interactive format. The website features a clean, modern design with smooth animations and user-friendly navigation, optimized for both desktop and mobile devices.

Purpose & Objectives:
Professional Presentation: Establish a strong online presence as a Computer Science student

Skill Demonstration: Showcase technical abilities in web development

Project Showcase: Display completed and ongoing projects with detailed descriptions

Contact Portal: Provide multiple channels for professional communication

Learning Platform: Document my growth journey and skills development
